// PROCESS.txt

#######
Domain
#######

ObjectInstance: Represents a specific object on the canvas.
PropertyInstance: Represents a property on an object instance.
CanvasObject: An object on the canvas, provided by the rendering library.
PropertySetter: A function that sets a property on a canvas object.

#######
Process
#######

---
GetRootObjects - Retrieves the root objects from the project.
--->

Object:
- object|earth
    - component|planet
        - property|velocity
        - object|circle
            - property|radius
    - object|moon
        - property|radius
- object|sun
    ...
...

--->
ExtractPropertyPaths - Extracts property paths from the object hierarchy.
--->

CloneCountPropertyPaths:
- object|earth > clone-count-property
- object|earth / object|moon > clone-count-property
- object|earth / component|planet / object|circle > clone-count-property

NonCloneCountPropertyPaths:
- object|earth / object|moon > property|radius
- object|earth / component|planet > property|velocity
- object|earth / component|planet / object|circle > property|radius

--->
AnnotateCloneCounts - Annotates property paths with clone counts.
--->

    CloneCountResults:
    - object|earth > clone-count-property: 10
    - object|earth / object|moon > clone-count-property: 5
    - object|earth / component|planet / object|circle > clone-count-property: 15

AnnotatedPropertyPaths:
- object|earth: (10) / object|moon: (5) > property|radius
- object|earth: (10) / component|planet > property|velocity
- object|earth: (10) / component|planet / object|circle: (15) > property|radius


--->
ExpandInstancePaths - Converts PropertyPaths to PropertyInstancePaths by expanding clone counts.
--->

earth: 1->10
    circle: 1->15

...
PropertyInstancePath: (object|earth: 5 / component|planet / object|circle: 10 > property|x)
PropertyInstancePath: (object|earth: 5 / component|planet / object|circle: 10 > property|radius)
PropertyInstancePath: (object|earth: 5 / component|planet / object|circle: 11 > property|x)
PropertyInstancePath: (object|earth: 5 / component|planet / object|circle: 11 > property|radius)
PropertyInstancePath: (object|earth: 6 / component|planet / object|circle: 1 > property|x)
PropertyInstancePath: (object|earth: 6 / component|planet / object|circle: 1 > property|radius)
...

--->
EvaluatePropertyInstances - Evaluates property instance paths to compute their corresponding values.
--->

...
PropertyInstanceResult: (object|earth: 5 / component|planet / object|circle: 10 > property|x, 100)
PropertyInstanceResult: (object|earth: 5 / component|planet / object|circle: 10 > property|radius, 25)
PropertyInstanceResult: (object|earth: 5 / component|planet / object|circle: 11 > property|x, 125)
PropertyInstanceResult: (object|earth: 5 / component|planet / object|circle: 11 > property|radius, 50)
...
...

--->
GroupObjectInstances - Nests property results into their object instances.
--->

ObjectInstanceResult:
- (property|x: 100)
- (property|radius: 25)

ObjectInstanceResult:
- (property|x: 125)
- (property|radius: 50)

--->
AcquireCanvasObjects - Acquires CanvasObjects for each object instance.
--->

CanvasObjectResult:
- (canvasObject: 1)
- (canvasObjectResult)
    - (property|x: 100)
    - (property|radius: 25)

CanvasObjectResult:
- (canvasObject: 2)
- (canvasObjectResult)
    - (property|x: 125)
    - (property|radius: 50)

--->
ExtractCanvasProperties - Extracts individual properties from library canvas objects for further processing.
--->

CanvasPropertyResult:
- (canvasObject: 1)
- (property|x: 100)

CanvasPropertyResult:
- (canvasObject: 1)
- (property|radius: 25)

CanvasPropertyResult:
- (canvasObject: 2)
- (property|x: 125)

CanvasPropertyResult:
- (canvasObject: 2)
- (property|radius: 50)

--->
MapPropertySetters - Maps extracted properties to their corresponding setter functions.
    - needs GetPropertySetter(property)
--->

PropertySetter:
- (setter: setX)
- (canvasObject: 1)
- (value: 100)

PropertySetter:
- (setter: setRadius)
- (canvasObject: 1)
- (value: 25)

PropertySetter:
- (setter: setX)
- (canvasObject: 2)
- (value: 125)

PropertySetter:
- (setter: setRadius)
- (canvasObject: 2)
- (value: 50)

--->
ExecutePropertySetters - Applies the setter functions to update the properties on library canvas objects.
---
